# Self Driving Car (In Progress)

## Overview

This project aims to develop an autonomous self-driving vehicle using Raspberry Pi, ultrasonic sensors, servo motors, and computer vision techniques.

The project is divided into two phases:

### Phase 1: Hardware Integration 

- Chassis Assembly
- Raspberry Pi Setup
- Motor Driver Integration
- Power Management
- Ultrasonic Sensor Mounting
- Servo Motor Integration

### Phase 2: Software Development 

- Obstacle Detection
- Autonomous Navigation
- OpenCV Integration
- Lane Detection
- Object Detection
- Deep Learning Based Decision Making

## Current Status

Hardware prototype completed.
Software implementation in progress.
